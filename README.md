# dotfiles

Sway desktop configuration for Surface Laptop 4.

## Contents

- `.config/sway/config` — window manager
- `.config/waybar/` — status bar
- `.config/foot/foot.ini` — terminal

## Dependencies

```
pacman -S sway waybar foot wmenu swaylock swayidle waybar playerctl
```

## Font

[Iosevka Oui](https://williamjansson.com/files/rice/dots/fonts/TTF-Unhinted/) — place TTFs in `~/.local/share/fonts/IosevkaOui/` and run `fc-cache -f`.
